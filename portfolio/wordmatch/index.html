Редактор и тренажер WordMatch

  • Категория: Туториалы
  • Дата публикации: 2022-08-08
  • Опубликовано на сайте: Библиотека программиста

Пошаговый туториал по разработке приложения WordMatch. Приложение состоит из трех модулей:

  1. Редактор пользовательских словарей.
  2. Тренажер для запоминания иностранных слов.
  3. Скрипт для создания базы данных.

Редактор и тренажер используют Tkinter в качестве GUI для SQLite.

Исходный код и скриншоты проекта - в репозитории WordMatch.
